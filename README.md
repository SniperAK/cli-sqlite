# cli-sqlite

Very simple sqlite usage on command line interface

# install 

```
npm install cli-sqlite -g;
```

# Usqage

## global sqlite ( local database )
```
$ sqlite 
```

## specific file 
```
$ sqlite db.sqlte
```

# special commands 

## !tables 
> show all tables ( aka `show tables` in mysql )
```
 db > !tables
```

 db > !tables

| (index) | name |
|-|-|
| 0 | 'test' |

> exit
